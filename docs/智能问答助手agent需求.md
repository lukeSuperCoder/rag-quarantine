# 智能问答助手agent需求
## 技术栈
- 前端：Vue3、Element Plus、Vite
- 后端：Python + Flask
- 数据库：Milvus
## 功能需求
1. 实现一个基于RAG技术的智能问答助手,结合向量数据库中的知识库内容来进行回答
2. 支持文档上传DOC文档内容提取转TXT,以及调用embedding模型完成向量嵌入
3. 支持重排序以及模型问答功能
4. 支持多轮对话功能,并且每次都应将对话内容加入上下文中，实现历史对话记忆功能
5. 支持知识库内容管理功能,支持添加、删除、修改知识库内容
6. 支持知识库内容查询功能,支持通过关键词搜索知识库内容
7. 设计前端对话流式传输，和后端相关功能联调
8. 回复的结果中，如果如果涉及到知识库内容，要能够声明出引用的文章内容和对应的具体段落
## 功能参考
1. scripts 这个脚本目录文件夹下已经基本实现了后端需要的相关功能，参考现有的功能来设计开发
2. 要支持markdown文档的渲染
3. 前端页面的设计风格要简约但是有科技风，有美感
4. 你的目标是开发一个生产级的供用户使用的系统。因此要减少很多说明、提示内容的展示，应该更注重于功能的展现
